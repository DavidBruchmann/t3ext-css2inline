=======================================
CSS to inline converter for direct mail
=======================================

.. only:: html

    :Classification:
        css2inline

    :Version:
        |release|

    :Language:
        en

    :Description:
        Moves the styles from CSS sheets into inline CSS, in order to comply with uncooperative
        email clients. Can be used as a post-processing stdWrap in TypoScript.
